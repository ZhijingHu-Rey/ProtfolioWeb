What I achieved?

This poject is the final project assignment of introduction to Self Driving Cars course offered by University of Toronto on Coursera. And more helpful resources could be found on Internet for preparing and self-learning.

In this project, the controller programming are written in Python to realize a self-driving car, which can autonomously drive around a track in Carla Simulator. The output values of the controller will be the vehicle throttle, brake and steering angle commands. The throttle and brake values come from the Longitudinal speed control, and the steering comes from the Lateral Control.

For the vehicle longitudinal control, a PID controller was selected. PID controller is easy implementation and tune, and could provides results in a wide range of systems. For the vehicle lateral control, the Stanley controller was selected due to its great performance and the huge amount of information found.

To realize the controller functionality, firstly, using the front axle coordinates and the closest waypoint coordinates to the vehicle to calculate the Cross track error. Then to estimate the Heading error by calculating the current angle of the road, using the current and the next waypoint.

Later, substracting the current yaw angle of the vehicle to obtain the Heading error is necessary. Finally, to find the total steering angle by adding up both the Cross track and the Heading errors.

To know more about the details, please check my GitHub page of this project: Self-Driving Vehicle Control.
